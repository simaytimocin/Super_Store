# Super Store

Super Store, genellikle ev eşyaları, giysiler ve elektrikli eşyaların yanı sıra yiyecek satan çok 
büyük bir süpermarkettir. Super Store, genellikle daha küçük emsallerinden yüzde 15 ila 45 
arasında daha az ücret alır._

Projede kullanılacak Super Store veri kümesine ait:

**Ship Mode** = Müşteri tarafından belirtilen nakliye şekli.

**Segment** = Müşterinin ait olduğu segment. 

**Country** = Müşterinin ikamet ettiği ülke.

**City** = Müşterinin ikamet ettiği şehir.

**Quantity** = Miktar

**Discount** = İndirim

**Profit** = Kâr

**State** = Eyalet

**Postal Code** = Posta Kodu

**Region** = Bölge

**Category** = Kategori

**Sub-Category** = Alt Kategori

Bu projede; sahip olduğumuz Super Store veri kümesi ile ilgili çeşitli analizler yapacağız. Bu 
analizler neticesinde ilgili mağazanın kârını nasıl artırabileceğimizi bulmayı amaçlıyoruz.
